# PHASE 1 DATA PACKAGE

This folder contains the complete data and documentation package for Phase 1 of the HPAI Modelling Challenge. All participating teams receive the same materials. 

These files define the epidemic context observed data streams for the period, player datasets, and include a template for team responses.

## Read this first

Before working with the data, please read: READ_FIRST-Phase1_narrative.pdf

## Folder contents

### Documents
• READ_FIRST-Phase1_narrative.pdf
    - This document provides the situational context, team objectives, and reporting instructions for the phase.
• DataDoc_Players.pdf
    - Describes the structure of each dataset provided.
• Kickoff_meeting_presentation.pdf
    - Copy of the presentation used in the kickoff meeting

### Epidemic scenario files
• activity.csv
    - Ledger of individual farm activity through the end of this phase.
• cases.csv
    - Observed cases through the end of this phase.
• mortality_ledgers.zip
    - Zip file containing farm-level mortality records, provided as daily ledgers, of 3 example farms.
• movement.csv
    - Animal movement records through the end of this phase.
• prev_culls.csv
    - Records of preventive culling events that occurred through the end of this phase.
