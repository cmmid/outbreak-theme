# Canonical player data

This folder contains the fixed reference datasets used throughout the HPAI Modelling Challenge.

These files define the shared epidemic setting (farms, geography, risk zones) and do not change across phases.

## Contents

• README.txt
  - The document that you are currently reading, thanks for taking the time to do so!
• Poultry_population_narrative.pdf
  - Population narrative
• population.csv
  - Player-facing farm population data (coordinate locations, production types, etc)
• counties_32626.geojson
  - County-level (i.e. NUTS 3) administrative boundaries (EPSG:32626)
• districts_32626.geojson  
  - District-level (i.e. LAU) administrative boundaries (EPSG:32626)
• clc_32626.geojson
  - Corine landcover of wetlands and water bodies (EPSG:32626)
• hrz_32626.geojson
  - High-risk zone polygon, (EPSG:32626)
